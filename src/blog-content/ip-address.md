## IP Address

Every device connected to the internet has a unique identity.

That identity is called an **IP Address**.

Just like your home has a postal address to receive letters, your device has an IP address to send and receive data across the internet.

Without it — communication simply wouldn’t be possible.

When you visit a website, your request doesn’t magically reach the destination.

It travels using:

- Your IP address (sender)
- The website’s IP address (receiver)

This allows data to know:

👉 Where it came from  
👉 Where it needs to go  

For example, when you open a website:

Your device might have an IP like:


192.168.1.2


And the server might have something like:


142.250.183.196


Your request moves across networks using these identifiers — just like packages moving between cities using postal codes.

IP addresses exist in two main versions:

- **IPv4** → Short numeric format (like `192.168.1.1`)
- **IPv6** → Longer format created to support the growing number of devices

Every email sent, website opened, or video streamed happens because devices can locate each other using IP addresses.

In simple terms:

No IP address = No internet communication.
